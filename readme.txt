== Installation ==

1. Upload 'monster-gallery' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create Gutenberg gallery!


== Frequently Asked Questions ==

= How create slider? =
The slider is automatically generated when gallrey collumns = 1

= Как создать слайдер? =
Слайдер создается автоматически, когда количество колонок = 1

= How work grid collumns with photos? = 
This works that standart gallery WordPress 5.0 function. You create gallery -> Select count columns. (1 = slider; 2-7 = grid with ligtbox)

= Как работает сетка фотографий? =
Как стандартная функция галереи в Wordpres 5.0. Вы создаете галерею -> Выбираете количеств колонок. (1 = слайдер; 2-7 = сетка с лайтбоксом)
